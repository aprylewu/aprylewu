# Mingyang Wu

I am a Research Assistant at [MMLab, The Chinese University of Hong Kong](https://mmlab.ie.cuhk.edu.hk/), working on multimodal learning, video understanding, and practical systems for audio-visual data and evaluation.

This GitHub is where I put research code, experiments, and side projects that are useful in day-to-day work.

## Current Focus

- Audio-video understanding and multimodal reasoning
- Timestamp-aware data pipelines for video and language tasks
- Evaluation tooling and research infrastructure for multimodal systems
- Building small, useful interfaces around research workflows

## Selected Projects

- [aitimeline](https://github.com/aprylewu/aitimeline)  
  Conference timeline browser built with Next.js for tracking AI venues and deadlines.
- [CS203Bproject_IntelligentScissors](https://github.com/aprylewu/CS203Bproject_IntelligentScissors)  
  Interactive image segmentation course project implemented in Java.
- Ongoing work on audio-visual captioning, timestamp alignment, and multimodal data pipelines.
